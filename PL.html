Programming Languages
There are many different types of Programming Languages each with there own respective purposes. Some examples are as follows.
C# Scripting
C# is a very common programming language that is used for game development. There are also other similar types such as C++ and C
Java Script
Java is a very diverse language. It can be used to create many different types of programs. Some of the more common uses are for website design and game dev.
HTML & CSS
Html is a script mainly used to create websites. It is usually used with CSS which is mainly the ascetics whereas Html provides the framework. Can also be used in 2d game development.
Python
This programming language is commonly found in game dev and robotic systems design. Python is a great starter language and is quite popular.
